# Document Generator

## Usage

```bash
jsonschema -o schema.md schema.json
```

## Help

```bash
jsonschema help
```

## Support

- [ ] Title
- [ ] Description
- [ ] Properties
- Link
  - Request
    - [ ] Queries
    - Example
      - [ ] Header
      - Body
        - Multipart
          - [ ] Multi-files
          - [ ] Multi-files as array
  - Response
    - Rel
      - [ ] Instances
    - EncType
      - [ ] Null
    - Example response
      - [ ] Header
      - [ ] Body
